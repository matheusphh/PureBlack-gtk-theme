# Installation is easy!
1. Copy the folders (gtk-3.0 , gtk-4.0 and gnome-shell) for .themes/ in "/home/username/".
2. Install User Themes Extension 
3. Apply with EvolveCore or Gnome-Tweaks

Don't tested in other DE's, only Gnome 49
